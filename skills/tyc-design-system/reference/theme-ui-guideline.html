ตัวอย่าง — ฟอร์มบันทึกข้อมูล 2 คอลัมน์ (Form Pattern)

รูปแบบฟอร์มมาตรฐานสำหรับทุกระบบภายในองค์กร ใช้ข้อมูลพนักงานเป็นเนื้อหาสมมติประกอบเท่านั้น

⚠ รูปแบบอีเมลไม่ถูกต้อง กรุณาตรวจสอบอีกครั้ง
ใช้สำหรับติดต่อกรณีฉุกเฉินเท่านั้น
<div class="form-row">
  <div class="form-group">
    <label>ชื่อ-นามสกุล<span class="req">*</span></label>
    <input type="text" class="form-control" placeholder="เช่น สมชาย ใจดี">
  </div>
  <div class="form-group">
    <label>แผนก<span class="req">*</span></label>
    <select class="form-control">
      <option value="">เลือกแผนก</option>
      <option>ฝ่ายผลิต (Production)</option>
    </select>
  </div>
</div>

<!-- error state -->
<input type="email" class="form-control is-invalid">
<div class="error-msg">⚠ รูปแบบอีเมลไม่ถูกต้อง</div>

<!-- ทางเลือก: label ตัวพิมพ์ใหญ่ (พบใน wms จริง) — ประกอบจาก utility class ที่มีอยู่แล้ว -->
<label class="text-xs font-semibold text-muted uppercase tracking-wide">Field Name</label>

Dialog / Modal

เรียกใช้ผ่าน openDialog('id') / closeDialog('id') — คลิกนอกกล่องเพื่อปิดได้เช่นกัน

<button onclick="openDialog('myDialog')">เปิด Dialog</button>

<!-- วางไว้นอก .app-shell เสมอ -->
<div class="dialog-overlay" id="myDialog"
     onclick="if(event.target===this) closeDialog('myDialog')">
  <div class="dialog">
    <div class="dialog-header">
      <h3>หัวข้อ</h3>
      <button class="dialog-close" onclick="closeDialog('myDialog')">✕</button>
    </div>
    <div class="dialog-body">เนื้อหา</div>
    <div class="dialog-footer">
      <button class="btn btn-secondary" onclick="closeDialog('myDialog')">ยกเลิก</button>
      <button class="btn btn-primary">บันทึก</button>
    </div>
  </div>
</div>

<!-- ปิดได้ด้วย Esc หรือคลิกนอกกล่องเช่นกัน -->

Alert / Banner & Toast

Alert ใช้แจ้งเตือนแบบฝังในหน้า (persistent) ส่วน Toast ใช้แจ้งเตือนชั่วคราวมุมขวาบน

แจ้งให้ทราบ
ระบบจะปิดปรับปรุงวันเสาร์ที่ 20 นี้ เวลา 22:00–24:00 น.
บันทึกสำเร็จ
ข้อมูลพนักงานถูกบันทึกเรียบร้อยแล้ว
โปรดตรวจสอบ
มีพนักงาน 3 คนที่ยังไม่ได้อัปเดตข้อมูลติดต่อ
เกิดข้อผิดพลาด
ไม่สามารถบันทึกข้อมูลได้ กรุณาลองใหม่อีกครั้ง
Toast — เรียกด้วย showToast('success' | 'warning' | 'danger') 
<div class="alert alert-success">
  <svg>...</svg>
  <div><div class="alert-title">บันทึกสำเร็จ</div>ข้อความ</div>
  <button class="alert-close">✕</button>
</div>

<!-- Toast: มี #toastStack วางไว้นอก .app-shell แล้ว เรียกได้เลย -->
<button onclick="showToast('success','หัวข้อ','ข้อความ')">
  แสดง Toast
</button>

Tabs

โครงสร้าง: .tabs ครอบ .tab-btn หลายปุ่ม คู่กับ .tab-panel ที่มี id ตรงกับ data-tab

ชื่อ-นามสกุล, เบอร์โทร, ที่อยู่ปัจจุบัน และผู้ติดต่อฉุกเฉิน

แผนก, ตำแหน่ง, วันที่เริ่มงาน และหัวหน้างาน

สำเนาบัตรประชาชน, ใบรับรองการทำงาน และเอกสารอื่นๆ

<div class="tabs">
  <button class="tab-btn active" onclick="switchTab(this,'tab1')">แท็บ 1</button>
  <button class="tab-btn" onclick="switchTab(this,'tab2')">แท็บ 2</button>
</div>
<div id="tab1" class="tab-panel active">เนื้อหาแท็บ 1</div>
<div id="tab2" class="tab-panel">เนื้อหาแท็บ 2</div>

Breadcrumb & Tooltip

Tooltip เป็น CSS-only ผ่าน attribute data-tooltip ไม่ต้องใช้ JS

Breadcrumb
แดชบอร์ด / พนักงาน / แก้ไขข้อมูล
Tooltip — วาง attribute data-tooltip="..." บน element ใดก็ได้ สถานะระบบ ⓘ 
<nav class="breadcrumb">
  <a href="#">หน้าหลัก</a>
  <span class="sep">/</span>
  <span class="current">หน้าปัจจุบัน</span>
</nav>

<!-- Tooltip แบบ CSS-only ผ่าน attribute เดียว -->
<button data-tooltip="ข้อความอธิบาย">วางเมาส์ตรงนี้</button>

Card — รูปแบบต่างๆ

เพิ่มคลาสต่อท้าย .card ได้เลย: .card-outline, .card-hover, .card-media, .card-list, .card-ribbon

.card-outline (ไม่มีเงา)
แจ้งเตือนระบบ
ใช้เมื่อวางการ์ดซ้อนบนพื้นที่มีเส้นขอบอยู่แล้ว
.card-hover (คลิกได้)
เปิดดูรายละเอียด
ยกขึ้นเมื่อ hover — ใช้กับการ์ดที่คลิกไปหน้าอื่นได้
.card-ribbon (ป้ายมุม)
ฟีเจอร์ล่าสุด
ใช้ attribute data-ribbon="ข้อความ" กำหนดป้ายได้เอง
.card-media (การ์ดแนวนอน มีไอคอน/รูปด้านซ้าย)
รายงานยอดขายเดือนนี้
อัปเดตล่าสุด 5 นาทีที่แล้ว
คำขออนุมัติใหม่ 3 รายการ
รอดำเนินการจากหัวหน้าแผนก
.card-list (รายการภายในการ์ดเดียว — ใช้ทำ activity feed)
สมชาย ใจดี อนุมัติคำขอลา2 นาทีที่แล้ว
กัญญา รักไทย ส่งคำขอเบิกอุปกรณ์18 นาทีที่แล้ว
ระบบแจ้งเตือน: เครื่องจักร #4 หยุดทำงาน1 ชั่วโมงที่แล้ว
