| โฟลเดอร์ | ใช้เก็บอะไร |

## ⚡ เริ่มต้นใช้งาน (สำหรับสมาชิกใหม่)

```bash
git clone https://github.com/YOUR_ORG/pi-claude-md.git
cd pi-claude-md
```

## 📤 วิธี Push งานขึ้น

```bash
git checkout -b feature/ชื่อ-skill-ของคุณ
git add .
git commit -m "feat: เพิ่ม skill ชื่อ-skill"
git push origin feature/ชื่อ-skill-ของคุณ
# จากนั้นเปิด Pull Request บน GitHub
```

## 📥 วิธี Pull อัปเดตล่าสุด

```bash
git checkout main
git pull origin main
```
